Pengantar Data & Visualisasi

Apa itu Data?

Data adalah kumpulan informasi yang dapat diproses dan dianalisis. Data dapat berupa angka, teks, gambar, atau bentuk lainnya. Dalam pengolahan data, kita mengubah data mentah menjadi informasi yang bermakna.

Jenis-Jenis Data

  • Data Kuantitatif: Data berupa angka (contoh: usia, tinggi badan, harga)
  • Data Kualitatif: Data berupa kategori (contoh: warna, jenis kelamin, kota)
  • Data Diskrit: Data yang dapat dihitung (contoh: jumlah siswa)
  • Data Kontinu: Data yang dapat diukur (contoh: berat badan, suhu)

Statistik Deskriptif

Statistik deskriptif adalah metode untuk merangkum dan mendeskripsikan data:

  • Mean (Rata-rata): Jumlah semua nilai dibagi jumlah data
  • Median: Nilai tengah setelah data diurutkan
  • Modus: Nilai yang paling sering muncul
  • Range: Selisih antara nilai maksimum dan minimum

Pengolahan Data dengan Python

Library Python untuk Data

Pandas: Library untuk manipulasi dan analisis data

import pandas as pd # Membuat DataFrame data = { 'Nama': ['Ani', 'Budi', 'Citra', 'Doni'], 'Nilai': [85, 90, 78, 92] } df = pd.DataFrame(data) # Menghitung statistik print(df['Nilai'].mean()) # Rata-rata print(df['Nilai'].median()) # Median print(df['Nilai'].max()) # Nilai maksimum

Contoh Analisis Data Sederhana

import numpy as np # Data nilai siswa nilai = [75, 80, 85, 90, 78, 88, 92, 76, 84, 87] # Statistik deskriptif mean = np.mean(nilai) median = np.median(nilai) std = np.std(nilai) print(f"Rata-rata: {mean}") print(f"Median: {median}") print(f"Standar Deviasi: {std}")

Visualisasi Data

Jenis-Jenis Grafik

  • Bar Chart: Membandingkan data kategori
  • Line Chart: Menampilkan tren data dari waktu ke waktu
  • Pie Chart: Menampilkan proporsi bagian dari keseluruhan
  • Scatter Plot: Menampilkan hubungan antara dua variabel
